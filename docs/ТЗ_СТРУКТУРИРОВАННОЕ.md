# Техническое задание
## Telegram‑бот для мотоциклистов (Екатеринбург)

*Структурированный документ с таблицами и примерами текстов сообщений бота.*

---

## 1. Общие сведения

| Параметр | Значение |
|----------|----------|
| Название проекта | Telegram‑бот для мотоциклистов |
| Город (этап 1) | Екатеринбург |
| Платформа | Telegram (далее — MAX) |

**Цели системы:**
- Объединить мото‑сообщество города
- Обеспечить быстрый обмен информацией: SOS, поиск мотопары, мероприятия, полезные контакты
- Реализовать платную подписку и платные сервисы (создание мероприятий, поднятие анкет)

---

## 2. Роли и права

### 2.1. Таблица ролей

| Роль | Описание |
|------|----------|
| **Суперадмин** | Владелец/главный администратор. Управляет всеми городами и настройками. |
| **Администратор** | Модератор конкретного города. |
| **Пилот** | Мотоциклист. Пользуется ботом, создаёт/посещает мероприятия. |
| **Двойка** | Пассажир. Пользуется ботом, ищет пилота, создаёт/посещает мероприятия. |

### 2.2. Права Суперадмина

| № | Функция |
|---|---------|
| 1 | Блокировка / разблокировка любых пользователей |
| 2 | Создание / редактирование администраторов (назначение по городу) |
| 3 | Повышение пользователя до администратора, понижение администратора до пользователя |
| 4 | Редактирование / удаление любых мероприятий (по всем городам) |
| 5 | Управление платной подпиской: включение / выключение, стоимость, длительность (месяц / сезон) |
| 6 | Гарантия: функция SOS работает всегда, независимо от статуса подписки пользователя |
| 7 | Создание рекламных объявлений (записи в блок «Полезные контакты» по любому городу) |
| 8 | Массовая рассылка пользователям (сегментация по городу, роли, статусу подписки) |
| 9 | Включение / выключение платного создания мероприятий, задание стоимости |
| 10 | Включение / выключение платной опции «поднять анкету» и задание стоимости |
| 11 | Просмотр статистики и логов (активность, подписки, SOS, мероприятия) |
| 12 | Ручное продление или отключение подписки конкретным пользователям (бонус/штраф) |
| 13 | Управление глобальными текстами: раздел «О нас», шаблоны уведомлений, системные сообщения |
| 14 | Управление городами (на будущее): добавление новых городов, назначение админов по городам |

### 2.3. Права Администратора города

| № | Функция |
|---|---------|
| 1 | Блокировка пользователей своего города (с обязательным уведомлением суперадмину с причиной) |
| 2 | Редактирование / удаление мероприятий своего города |
| 3 | Создание рекламных объявлений по своему городу (записи в «Полезные контакты») |
| 4 | Просмотр жалоб пользователей на анкеты/мероприятия своего города |
| 5 | Пометка мероприятий как «рекомендуемые» / «официальные» (отображать визуально) |
| 6 | Временное скрытие анкеты пользователя из поиска (soft‑ban) без полной блокировки аккаунта |

### 2.4. Права Пилота

| № | Функция |
|---|---------|
| 1 | Создание / редактирование своей анкеты (кроме номера телефона) |
| 2 | Создание / редактирование / удаление своих мероприятий (с учётом платности и подписки) |
| 3 | Просмотр анкет в блоке «Мотопара» и использование лайков/дизлайков |
| 4 | Участие во взаимных лайках, получение контактов при совпадении |
| 5 | Просмотр мероприятий и запись на мероприятия |
| 6 | После записи на мероприятие — установка статуса «Ищу двойку» и работа с запросами от двоек |

### 2.5. Права Двойки

| № | Функция |
|---|---------|
| 1 | Создание / редактирование своей анкеты (кроме номера телефона) |
| 2 | Создание / редактирование / удаление своих мероприятий (если бизнес‑логикой не ограничено) |
| 3 | Просмотр анкет в блоке «Мотопара» и использование лайков/дизлайков |
| 4 | Участие во взаимных лайках, получение контактов при совпадении |
| 5 | Просмотр мероприятий и запись на мероприятия |
| 6 | После записи на мероприятие — установка статуса «Ищу мотоциклиста» и работа с запросами пилотов |

## 3. Регистрация и анкеты

### 3.1. Общий сценарий

1. Пользователь запускает бота (`/start`)
2. Бот предлагает выбрать город (на первом этапе — только «Екатеринбург»)
3. Выбор роли: **Пилот** или **Двойка**
4. Заполнение анкеты по выбранной роли

### 3.2. Анкета Пилота

| Поле | Тип / формат | Обязательное | Особенности |
|------|--------------|--------------|-------------|
| Имя / никнейм | Текст | Да | Кликабельная ссылка на username Telegram |
| Телефон | Контакт Telegram | Да | Только кнопка «Отправить номер» |
| Возраст | Число (лет) | Да | 18–80 |
| Пол | Муж / Жен / Другое | Да | |
| Марка мотоцикла | Текст | Да | |
| Модель мотоцикла | Текст | Да | |
| Кубатура | Число (см³) | Да | |
| Стаж вождения | Дата (ДД.ММ.ГГГГ) | Да | В БД — дата, при выводе — стаж в годах |
| Стиль вождения | Спокойный / Агрессивный / Смешанный | Да | |
| Фото | Фото | Нет | |
| О себе | Текст | Нет | До 500–1000 символов |

### 3.3. Анкета Двойки

| Поле | Тип / формат | Обязательное | Особенности |
|------|--------------|--------------|-------------|
| Имя / никнейм | Текст | Да | Кликабельный username |
| Телефон | Контакт Telegram | Да | Только кнопка |
| Пол | Выбор | Да | |
| Возраст | Число (лет) | Да | |
| Вес | Число (кг) | Да | |
| Рост | Число (см) | Да | |
| Желаемый стиль | Спокойный / Динамичный / Смешанный | Да | |
| Фото | Фото | Нет | |
| О себе | Текст | Нет | |

**Общее правило:** Все поля, кроме телефона, редактируются в «Мой профиль». Телефон меняется только через поддержку/админа.

## 4. Блок «SOS»

**Подкнопки:** ДТП | Сломался | Обсох | Другое

**Флоу:**
1. Выбор типа SOS
2. Обязательная геолокация
3. Опциональный комментарий
4. Рассылка всем пользователям города

**Ограничение:** не чаще 1 раза в 10–15 минут (настраиваемый параметр).

---

## 5. Блок «Мотопара»

- **Меню:** Анкеты Пилотов | Анкеты Двоек → «Все анкеты» | «Фильтр»
- **Фильтр:** пол, возраст до N, вес до N, рост до N (опционально)
- **Просмотр:** по одной анкете, кнопки Лайк | Дизлайк | Следующая
- **Дизлайк x3:** взаимное скрытие анкет
- **Лайк:** уведомление владельцу → «Посмотреть анкету» → Лайк/Дизлайк в ответ
- **Взаимный лайк:** раскрытие контактов, сообщение «Совпадение», кнопка «Перейти в чат»
- **Поднятие анкеты:** платная опция (если включена)

---

## 6. Блок «Полезные контакты»

**Категории:** МотоМагазин | МотоСервис | МотоШкола | МотоКлубы | МотоЭвакуатор | Другое

**Поля:** название, категория, описание, телефон/ссылка/сайт, адрес (опционально), город.

**Кто добавляет:** суперадмин (все города), администратор города (свой город).

---

## 7. Блок «Мероприятия»

**Типы:** Масштабные | Мотопробеги | Прохваты

**Функции:** создать | просмотреть + записаться | мои мероприятия

**После записи:** «Ищу двойку» / «Ищу мотоциклиста» — поиск пары в рамках мероприятия.

---

## 8. Блок «Мой профиль»

- Просмотр и редактирование анкеты (кроме телефона)
- Статус подписки
- Подключение/продление подписки
- Поднять анкету

---

## 9. Блок «О нас»

- Описание проекта
- Контакты (email, Telegram поддержки)
- Кнопка «Написать в поддержку»
- Форма «Предложения и пожелания»
- Кнопка «Поддержать проект» (донат)

---

## 10. Примеры текстов сообщений бота

### 10.1. Онбординг

**При /start (новый пользователь):**

```
Привет! 👋
Это бот мото‑сообщества Екатеринбурга.

Здесь ты можешь:
• 🚨 Отправить SOS в экстренной ситуации
• 🏍 Найти мотопару
• 📇 Узнать полезные контакты (сервисы, магазины, эвакуаторы)
• 📅 Создавать и посещать мероприятия

Для начала выбери город и свою роль.
```

**Кнопки:** [Екатеринбург]

**Далее:** [Я Пилот] [Я Двойка]

---

### 10.2. Заполнение анкеты

**Выбор роли:**
```
Отлично! Ты выбрал роль Пилота.
Давай заполним короткую анкету, чтобы другие знали, кто ты и как к тебе обратиться.
```

**Имя:**
```
Введи своё имя или никнейм.
Мы покажем его другим вместе с твоим Telegram‑логином.
```

**Телефон:**
```
Теперь отправь свой номер телефона кнопкой ниже.
Вводить номер вручную нельзя — только через Telegram.
```
**Кнопка:** [Отправить мой номер]

---

### 10.3. Уведомление о лайке

```
Твою анкету оценили 👍
Нажми «Посмотреть анкету», чтобы увидеть, кто поставил лайк.
```
**Кнопка:** [Посмотреть анкету]

**После просмотра:**
```
Нравится этот человек?
Поставь лайк, и если симпатия взаимная — вы получите контакты друг друга.
```
**Кнопки:** [Лайк] [Дизлайк]

---

### 10.4. Взаимный лайк

```
Совпадение! 💥
Вы понравились друг другу.
Контакты: [ссылка/username]

Желаем безопасных и приятных поездок! 🏍
```
**Кнопка:** [Перейти в чат]

---

### 10.5. SOS‑уведомление

```
🚨 SOS: ДТП

Имя: Иван (@ivan_moto)
Телефон: +7 900 123‑45‑67
Комментарий: «Упал на Горького, нужна помощь с эвакуатором.»

📍 Местоположение (кнопка с геолокацией)
```
**Кнопки:** [📞 Позвонить] [✈️ Написать в Telegram]

---

### 10.6. Сообщение о подписке

**При попытке открыть Мотопару без подписки:**

```
Для доступа к поиску мотопары и контактам анкет нужна активная подписка.

Подписка даёт:
• Просмотр анкет пилотов и двоек
• Лайки и совпадения с контактами
• Создание мероприятий бесплатно
• Поднятие анкеты (по настройке)

Выбери срок:
```
**Кнопки:** [1 месяц — XXX ₽] [Сезон — YYY ₽] [Назад]

### 10.7. Ошибки

| Ситуация | Пример текста |
|----------|---------------|
| Общая ошибка | Произошла ошибка. Попробуй позже. |
| Заблокирован | Вы заблокированы. Обратитесь в поддержку. |
| Нет города | Ошибка: город не выбран. Нажми /start |
| Cooldown SOS | Следующий SOS доступен через X мин. Y сек. |
| Ошибка сохранения | Ошибка при сохранении. Попробуй /start и пройди регистрацию заново. |

## 11. Логика подписки

| Подписка в настройках | Пользователь | Доступ |
|-----------------------|--------------|--------|
| Выключена | Любой | Всё бесплатно (кроме отдельных платных фич) |
| Включена | Без подписки | Только SOS и «Полезные контакты» |
| Включена | С подпиской | Полный доступ, создание мероприятий бесплатно |

**Гарантия:** SOS доступен всегда, независимо от подписки.
