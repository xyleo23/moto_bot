# Диагностика: бот перестал отвечать после настройки ролей

## Заключение

**Причина:** обработчики админ-клавиатуры при отсутствии прав выполняют **молчаливый `return`** без ответа. Пользователь с админ-клавиатурой, утративший права, не получает ответа ни на одну кнопку, в том числе на «🏠 Главное меню».

---

## Детальный разбор

### Сценарий

1. Пользователь с ID в `SUPERADMIN_IDS` заходил в `/admin` и получал админ-клавиатуру.
2. После изменения настроек ролей (правка `.env`, смена суперадмина и т.п.) его ID больше не считается суперадмином.
3. Админ-клавиатура **остаётся** (она persistent в Telegram).
4. При нажатии любой кнопки вызывается соответствующий обработчик, но из‑за проверки прав выполняется `return` без вызова `message.answer()`.
5. Пользователь не получает ответа — бот «молчит».

### Проблемные обработчики в `admin.py`

| Обработчик | Условие | Результат |
|------------|---------|-----------|
| `msg_admin_stats` (📊 Статистика) | `if not _is_superadmin: return` | Молчаливый выход |
| `msg_admin_users` (👥 Пользователи) | то же | то же |
| `msg_admin_city_admins` (🏙 Админы городов) | то же | то же |
| `msg_admin_settings` (⚙️ Настройки) | то же | то же |
| `msg_admin_broadcast` (📢 Рассылка) | то же | то же |
| `msg_admin_contacts` (📇 Контакты) | `if not can_manage: return` | то же |
| `msg_admin_text_about` (📝 О нас) | `if not _is_superadmin: return` | то же |
| **`msg_admin_main_menu` (🏠 Главное меню)** | `if not is_sa and not is_ca: return` | **Блокируется выход в меню** |

Особенно важно: «🏠 Главное меню» — это кнопка выхода из админки. При потере прав она тоже не срабатывает, и пользователь «застревает» на админ-клавиатуре.

### Исключение

- **«📅 Мероприятия»** обрабатывается в `start.kb_events` и корректно отвечает и админам, и обычным пользователям.

---

## Решение

1. **Кнопка «🏠 Главное меню»** должна **всегда** возвращать в главное меню, независимо от прав. Это escape-кнопка.
2. Желательно добавить fallback для не-админов при нажатии остальных кнопок админ-клавиатуры (сообщение «Нет доступа» или возврат в главное меню).
